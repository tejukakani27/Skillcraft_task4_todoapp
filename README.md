# SkillCraft Task 04 - To-Do Web Application

This project was developed as part of the SkillCraft Technology Web Development Internship.

## Project Overview

The To-Do Web Application helps users efficiently manage daily tasks by allowing them to add, edit, delete, organize, and track task completion with date and time scheduling.

## Features

- Add New Tasks
- Edit Existing Tasks
- Delete Tasks
- Mark Tasks as Completed
- Set Date and Time
- Task Organization
- Local Storage Support
- Responsive Design
- User-Friendly Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Learning Outcomes

- DOM Manipulation
- CRUD Operations
- Local Storage Implementation
- Event Handling
- Dynamic Content Management
- Responsive Web Design
